# Library API

FastAPI приложение для управления библиотечными книгами с интеграцией Apache Kafka.

## Описание

Это приложение предоставляет REST API для управления книгами в библиотеке. Оно позволяет:
- Получать список всех книг
- Получать информацию о конкретной книге
- Создавать новые книги
- Обновлять информацию о книгах
- Удалять книги
- Поиск книг по автору
- Интеграция с Apache Kafka для асинхронной обработки сообщений

## Технологии

- FastAPI — фреймворк для создания веб-API
- SQLAlchemy — ORM для работы с базой данных
- SQLite — локальная база данных
- FastStream — фреймворк для работы с Apache Kafka
- Pydantic — валидация данных

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `uv pip install -e .`
3. Запустите приложение: `uv run app/main.py`

## Использование

После запуска приложение будет доступно по адресу http://localhost:8000

Документация к API доступна по адресу http://localhost:8000/docs

## Конфигурация

Приложение использует переменные окружения для конфигурации:
- `DATABASE_URL` — строка подключения к базе данных (по умолчанию: sqlite:///./library.db)

## Логирование

Приложение записывает логи в папку `logs`:
- `app.log` — логи приложения
- `sql_queries.log` — SQL-запросы
